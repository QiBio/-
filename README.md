## 注意事项
- 正确区分**工作目录**与**主目录**。**工作目录**是服务器管理员分配给每位用户的用户专属目录，用户可点击此处进行查看；**主目录**是用于存储用户配置文件的目录，可使用命令`echo $HOME`进行查看。受限于服务器的存储空间分配，用户仅能在**工作目录**下进行包括**配置生物分析环境**在内的所有生物信息分析工作，仅能在**主目录**下进行用户配置文件的修改。
